# Student Survey Analytics Dashboard

Interactive Streamlit dashboard for visualizing student satisfaction survey data from IIT Madras Zanzibar Campus. Loads data from Google Sheets, processes with pandas, and creates Plotly visualizations.

## Features
- Key metrics (Academic, Mess, Campus satisfaction)
- Radar chart & category comparison
- Demographics (gender, ethnicity, degree, major, graduation year)
- Campus feedback (liked aspects, needed improvements)
- Future plans (higher studies, job sectors, majors)
- Mess experience (menu, food quality, dietary preference)
- Club analysis (memberships, beneficial events, new club interest)
